Zum Hochladen eines Bildes klicken Sie auf den Button "Bild auswählen"

Nachdem Sie ein Bild hochgeladen haben, können Sie das Bild für andere Mitglieder freigeben.
Zusätzlich bestimmen Sie welches Bild in Ihrem Profil als Profilbild angezeigt wird.
Um ein Bild als Profilbild anzuzeigen, klicken Sie auf den Button "Profilbild".
Sie können nur ein Bild als Profilbild auswählen.
